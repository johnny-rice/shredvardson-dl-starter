# Prompt Library

Organized collection of LLM prompts for development workflows.

## Structure

- **system/** - Core behavioral prompts and guardrails
- **task/** - Specific development task templates  
- **tools/** - Tool-specific guidance and configurations

## Usage

All prompts follow a standard header format for LLM-friendly discovery and usage.

## Requirements

- Each prompt must include standardized header (see template)
- Directory READMEs provide navigation context
- No loose .md files outside organized directories
